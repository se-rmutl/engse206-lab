# Weekly Assignment Contract

Course Repo และ Team Repo ใช้สัญญารูปแบบเดียวกันในทุกสัปดาห์

## ฝั่ง Course Repo ต้องประกาศ

- Assignment ID / Version
- Case input ที่ใช้
- Learning outcomes
- Required steps
- Exact output paths
- Evidence requirements
- Rubric
- Deadline

## ฝั่ง Team Repo ต้องบันทึก

- Assignment URL / Version ที่อ่าน
- Case ID / Case URL
- Artefact links
- Evidence links
- Worklog และ contribution
- Peer/instructor feedback
- Submission commit hash
- Known limitations / open questions

## กติกา Version

- Course assignment: `W03-v1.0`
- Student work: commit message `submit(w03): ...`
- Milestone release แนะนำ: `discover-v1.0`, `srs-v1.0`, `design-review-v1.0`, `final-v1.0`
