# Week 04 Role Packs

## วิธีใช้

1. Copy เฉพาะ Role ที่กำลังจำลองไปยัง prompt template.
2. Use one chat/session per role.
3. Do not add facts outside Case Card and Role Pack.
4. Save prompt and anonymized notes in Team Repo evidence.
5. Tag all AI responses as simulation and assign confidence/follow-up.
